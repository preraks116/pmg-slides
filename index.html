Procedural Maze Generation

Objective of the Project


• To create a simple maze generating application/game.

• To be able to use different kinds of algorithms can be   used to generate the maze.

• To be able to replay old mazes using seeds.

• To be able quantify "difficulty" of any given maze.

• Allow user to check their performance on solving the     maze using parameters like path, time taken and           difficulty/complexity of the maze.

Methodology


• Creating a physics based game using ThreeJS and           CannonJS

• Created custom Object Oriented JS Wrappers.

• Different algorithms output the structure of the maze     in a common format

• As the player navigates the ball through the maze,       parameters like taken and path taken are stored
